spnr.str.symbols

Reference contents

Syntax:

spnr.str.symbols

Example:

var char = spnr.choose(spnr.str.symbols)

Description:

An array containing all of the symbols on my keyboard. In the example above, char would be a random symbol. All values: ['~', '`', '!', '@', '#', '$', '%', '^', '&', '*', '(', ')', '-', '_', '=', '+', '[', '{', ']', '}', '\\', '|', ';', ':', '\'', '"', ',', '<', '.', '>', '/', '?']
