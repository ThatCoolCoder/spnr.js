spnr.MouseWatcher.touchDown

Reference contents

Syntax:

spnr.MouseWatcher.touchDown

Example:

var canJump = watcher.mouseDown

Description:

Boolean that states whether the touchscreen is being touched in watcher.elem. Note: this only tracks the touchscreen, not mouse For a boolean responding to both mouse and touchscree, see spnr.MouseWatcher.pointerDown.
