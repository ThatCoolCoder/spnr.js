spnr.dom.logPara

Reference contents

Syntax:

spnr.dom.logPara

Example:

Description:

A paragraph element which can be used to log data to. Only created when spnr.dom.logToPara is called. Has access methods spnr.dom.clearLogPara and spnr.dom.logToPara
